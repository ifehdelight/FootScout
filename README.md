# FootScout 

A simple, mobile-first web app to search for soccer players using TheSportsDB API.

### Features
- Live player search by name
- Shows player photo, name, current club
- Filters out non-soccer results automatically  
- Responsive design for Android/iOS
- Built with vanilla HTML, CSS, and JavaScript - zero dependencies

### Tech Stack
- HTML5 + CSS3 + Vanilla JS
- TheSportsDB API v1
- Mobile-first responsive layout

### Limits
Uses TheSportsDB free tier: 100 requests/month. No stats, goals, or trophies - only basic player bio data.
